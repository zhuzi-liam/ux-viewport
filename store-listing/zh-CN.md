# Chrome Web Store 上架文案

## 基本信息

- 名称：UX Viewport
- 摘要：一键将 Chrome 或 Arc 的网页可视区调整到设计稿尺寸，适合 UX 验收与 1:1 对照。
- 类别：开发者工具
- 语言：简体中文
- 可见性：不公开（Unlisted）

## 详细说明

UX Viewport 面向 UX 设计师和前端团队，用于把真实浏览器窗口的网页可视区快速调整到指定尺寸，无需打开 DevTools 或手动拖动窗口。

- 内置 1728×1117、1440×900、1280×800、768×900
- 支持自定义尺寸、预置编辑和默认尺寸
- 实时显示当前网页可视区
- 悬浮控件可拖动，并自动吸附左右边缘
- 尺寸不改变页面缩放比例

扩展只在用户主动点击后作用于当前网页，所有设置仅保存在本机浏览器中。

## 单一用途说明

调整当前浏览器窗口，使网页可视区达到用户选择的尺寸，并显示当前可视区尺寸。

## 权限说明

- `activeTab`：用户点击扩展图标后，临时访问当前网页。
- `scripting`：向当前网页注入尺寸选择和测量控件。
- `storage`：在本机保存预置尺寸、默认项和控件会话状态。

## 数据使用声明

不收集或传输任何用户数据、网页内容、身份信息、分析数据或浏览历史。

## 审核测试步骤

1. 打开任意普通网页并点击 UX Viewport 图标。
2. 点击默认尺寸按钮，确认窗口可视区调整到 1440×900。
3. 打开下拉菜单测试其他预置、自定义尺寸和默认项。
4. 拖动悬浮控件，确认松手后吸附左右边缘。
